## ğŸš€ Getting Started

### Prerequisites

-   Node.js (v16+)\
-   npm or yarn package manager

### Installation

``` bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/AURA-VIBE.git

# Navigate to project folder
cd AURA-VIBE

# Install dependencies
npm install
```

### Run the Development Server

``` bash
npm run dev
```

### Build for Production

``` bash
npm run build
```
